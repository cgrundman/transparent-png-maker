# Transparent PNG Maker

Code to take pngs and make the backgrounds transparent

## Run Locally

Clone the project:

```bash
git clone https://github.com/cgrundman/transparent-png-maker.git
```

Go to the project directory:

```bash
cd local/path/transparent-png-maker
```

In the terminal:

```bash
python main.py
```
